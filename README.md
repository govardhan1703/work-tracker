# 📊 Daily Tracker

> A full-stack productivity tracking application to record, manage, and analyze daily activities efficiently.

---

## 🌐 Live Demo

- 🖥️ Frontend (UI): https://work-tracker-psi-gilt.vercel.app  
- ⚙️ Backend (API): https://work-tracker-1-3ckt.onrender.com  

⚠️ Note: Backend is an API server. Access endpoints like:
👉 https://work-tracker-1-3ckt.onrender.com/api/history/2026-04-05
---

## 📌 Features

* ✅ Add Daily Activities / Tasks
* ✅ View History of Daily Logs
* ✅ Local Cache Support (fallback if backend fails)
* ✅ Date-wise Tracking System
* ✅ Clean and Responsive UI
* ✅ REST API Integration

---

## 🛠️ Tech Stack

### Frontend

* React.js
* Tailwind CSS
* Vite

### Backend

* Node.js
* Express.js

### Database

* MongoDB (Mongoose)
